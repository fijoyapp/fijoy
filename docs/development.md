## Commands

We use ![mise](https://mise.jdx.dev/) to manage the development environment.
Activating mise will give you all the tools you need to work on this project.

### Database

```bash
just compose up
just compose down
```

### Server

```bash
just server
just codegen # Ent, GQL, etc.
```

### Web

```bash
just web
```

### GraphQL / Relay

```bash
just dev

```
