## Commands

### Database

```bash
just db-up
just db-down
```

### Server

```bash
just server
just codegen # Ent, GQL, etc.
```

### Web

```bash
just web
```

### GraphQL / Relay

```bash
just relay-watch
just merge-graphql

```
